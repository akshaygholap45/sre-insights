# Mintoak Marketing Website — UI Kit

High-fidelity recreation of the **Mintoak marketing site** — the public B2B site selling the platform to banks.

## Run
Open `index.html` (single-file React in `App.jsx`). Loads `../../colors_and_type.css` + Lucide CDN.

## Sections (`App.jsx`)
- `Nav` — sticky frosted header, **Products** mega-dropdown (hover), "Book a demo" CTA.
- `Hero` — Mint-Tint gradient, headline with green accent, product screenshot mock.
- `Trust` — bank logo strip · `Features` — 6-card grid · `Stats` — Forest Ink metric band.
- `CTASection` — tinted closing CTA · `Footer` — dark Forest, link columns.

## Notes
- Marketing voice is slightly more aspirational than product UI; copy is illustrative.
- CTAs use 12px rounded-rectangle radius; Lato throughout.
